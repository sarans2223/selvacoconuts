# **App Name**: Selva AgriExports

## Core Features:

- Product Catalog: Showcase Coir Pith (Cocopeat) and Coir Fibre (Thanga Naaru) products with detailed descriptions and images.
- Sustainability Showcase: Dedicated section highlighting the eco-friendly practices and certifications of Selva Adhitya Coirs and Pith Blocks.
- Interactive Global Map: Visualize the company's global reach and export destinations, offering a dynamic user experience.
- Contact Form with Geolocation: Allow the client to send request based on user's current geolocation.
- Intelligent Quote Tool: A tool to parse text entered in the quote request box, use it to identify which of the coir products are requested, and then generate an appropriate response.

## Style Guidelines:

- Primary color: Deep Forest Green (#228B22) to emphasize sustainability and nature.
- Background color: Desaturated Earthy Brown (#D2B48C), brightness adjusted to be appropriate for a light scheme.
- Accent color: A slightly brighter, desaturated yellow-green (#ADFF2F) to complement the primary color and create contrast.
- Font pairing: 'Playfair' (serif) for headlines to convey elegance and high quality, and 'PT Sans' (sans-serif) for body text for readability.
- Use clean, line-based icons to represent 'Biodegradable', 'Export Quality', 'Bulk Supply', and 'Rural Employment' benefits.
- Modern and responsive grid layout with clear sectioning to enhance user navigation and content presentation.
- Subtle transitions and animations for loading states and interactive elements to improve user engagement.
